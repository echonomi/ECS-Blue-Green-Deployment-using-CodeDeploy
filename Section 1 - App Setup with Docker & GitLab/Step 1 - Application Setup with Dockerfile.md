### Nest.js Installation 
To Install `Nest.js`, navigate to your terminal and run the command below:
```shell
npm i -g @nestjs/cli
```

![[Pasted image 20251111115035.png]]
### Create Your First Nest.js Application
To create a `Nest.js` application, navigate to your terminal and run the command below:
```shell
nest new <project-name>
```

![[Pasted image 20251111115125.png]]

### RUN Nest.js Application
To run the `Nest.js` application kindly run the command below

```shell
npm run start
```

![[Pasted image 20251111115615.png]]

Next, open your browser and navigate to `http://localhost:3000`
![[Pasted image 20251111115634.png]]
